# 🚛 Municipal Fleet Optimization & Maintenance Analytics

## 📌 Project Overview

The Municipal Fleet Optimization & Maintenance Analytics project is an end-to-end Data Analytics and Machine Learning solution designed to help municipal organizations efficiently manage government vehicle fleets.

The project combines MySQL, Python, Machine Learning, and Power BI to analyze fleet performance, maintenance history, vehicle health, operating costs, and future maintenance requirements. It enables municipal departments to make proactive, data-driven decisions that reduce maintenance costs, improve vehicle utilization, and extend fleet lifespan.

## 📖 Business Problem

Municipal organizations operate hundreds or thousands of vehicles across departments such as sanitation, public works, police, fire, and parks.

Common challenges include:
- Increasing maintenance costs
- Unexpected vehicle breakdowns
- Aging fleet assets
- Poor maintenance planning
- High vehicle downtime
- Budget inefficiencies

Traditional reporting only identifies issues after they occur. This project enables fleet managers to monitor fleet health, predict maintenance needs, and optimize fleet operations proactively.

## 🎯 Project Objectives
- Analyze municipal fleet performance
- Monitor vehicle health and degradation
- Evaluate maintenance costs and repair trends
- Identify high-cost vehicles
- Predict future maintenance requirements using Machine Learning
- Build interactive Power BI dashboards
- Generate actionable fleet management recommendations

## 🗂 Dataset Overview

The dataset contains fleet inventory, maintenance history, and machine learning prediction data.

| Category | Columns |

| Cost | Purchase Price, Maintenance Cost |

## 📈 Power BI Dashboards

**Dashboard 1 — Executive Fleet Dashboard**
Provides an executive summary of fleet performance. Includes KPI Cards, Department-wise Vehicles, Vehicle Status Distribution, Fleet Health, Maintenance Cost, Vehicle Type Analysis.

**Dashboard 2 — Vehicle Dashboard**
Monitors vehicle condition and utilization. Includes Vehicle Type Breakdown, Fuel Type Distribution, Purchase Price vs Odometer, Fleet Degradation (Current vs Future), Vehicle Details, Department & Status Slicers.

**Dashboard 3 — Maintenance Dashboard**
Analyzes maintenance activities and repair costs. Includes Total Repairs, Maintenance Cost Trend, Downtime Analysis, Maintenance Type Distribution, Maintenance Code Frequency, Repair Details Table.

## 📌 Key Business Insights

- Older vehicles generally have higher maintenance costs.
- Certain departments operate aging fleets requiring replacement planning.
- Preventive maintenance reduces unexpected downtime.
- Fleet degradation scores help prioritize vehicle servicing.
- High-mileage vehicles contribute significantly to maintenance expenses.

## 💼 Business Recommendations

- Schedule preventive maintenance for high-degradation vehicles.
- Replace aging vehicles with consistently high repair costs.
- Allocate maintenance budgets based on department-wise fleet condition.
- Monitor vehicle health using degradation scores.
- Prioritize servicing for high-mileage vehicles before breakdowns occur.
